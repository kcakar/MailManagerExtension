How to use?

  1. Go to db
  2. Execute below query
    • SELECT TOP (1000) 
    Portal.Name	as PortalName
    ,Portal.HttpLink
    ,Campaign.Name as CampaignName
    ,Content.AppId
    ,Portal.PortalId
    ,[Description]
    ,[Title]
    ,ContentId
    ,[Content]
    ,Content.Id
    ,[LayoutContentId]
    ,Content.Id

FROM [ContentManager].[dbo].[ContentItems] Content 
FULL JOIN [TMA_HandyAnkauf_WebService].[dbo].[TradeInConfig] Config ON Content.AppId=Config.AppId
FULL JOIN TMA_Teqport_Config.dbo.Portal Portal ON Portal.PortalId=Config.PortalId
FULL JOIN TMA_HandyAnkauf_WebService.dbo.Campaign Campaign ON Config.CampaignId = Campaign.Id 
WHERE Content IS NOT NULL
ORDER BY Portal.Name,Content.LayoutContentId
  3. Select all results, right click and save as CVS file
  4. Choose CVS file
